# My Website

Welcome to my files! Follow these simple steps to view my website:

1. Click the **Code** icon.
2. Select **Download ZIP** (it’s only 10.3 MB).
3. Wait for the download to complete.
4. Once finished, open the downloaded ZIP file.
5. Extract the file using **WinRAR** or your computer’s built-in extractor.
6. After extraction, open the newly created folder.
7. Open the **My-website-main** folder.
8. Find and double-click **Home.html** (use the search bar if you can’t locate it).
9. That’s it! Enjoy exploring my website!
